\# Sistema de Leilões



\*\*Descrição do Projeto:\*\*

Este projeto é um sistema informatizado desenvolvido para gerenciar e automatizar as rotinas de uma casa de leilões.
De forma simples, o sistema organiza as vendas da casa por meio de uma tabela onde o usuário pode rapidamente consultar e vender os produtos.



\*\*Tecnologias Utilizadas:\*\*

\* Java

\* MySQL
